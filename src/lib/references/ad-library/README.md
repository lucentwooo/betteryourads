## How to use

During the Strategy phase of `ad-creative-generator-v2`:
1. Identify the brief's awareness stage and register.
2. Pull 2-3 matching analyses from this library.
3. Reference the specific patterns in `_patterns.md` they exemplify.
4. Design the creative to echo those patterns without copying specific imagery.

Cross-brand patterns distilled in [`_patterns.md`](./_patterns.md).
